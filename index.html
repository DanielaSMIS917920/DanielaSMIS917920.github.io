Introducción

Este portafolio web es realizado como instrumento de evidencia del proceso, aprendizaje y practicas de la estudiante Daniela Alvarez durante el Ciclo IV en la materia de Programación Computacional IV.

Descripción de la asignatura

Los estudiantes comprenden y aplican los lenguajes y tecnologías para el desarrollo de aplicaciones para computadoras, dispositivos móviles; sin embargo, aún no conocen la integración de los elementos de desarrollo de sistemas para la Web que les permita crear soluciones profesionales en entornos distribuidos. Se pretende que, en la asignatura, los estudiantes adquieran las competencias para desarrollar programas orientados a entornos Web.

Contenidos de la asignatura

Primer Computo:

- Tema Semana 1: Introducción a las librerias Bootstrap y VueJS.

- Tema semana 2: Rendering y directivas con Vue.

- Tema semana 3: Eventos: Clases y estilos.

- Tema semana 4: Componentes.

- Tema semana 5: Comunicación entre componentes.


Segundo Computo:

- Tema semana 7: Introducción al desarrollo con PHP.

- Tema semana 8: Introducción a laravel: Conexión a una base de datos.

- Tema semana 9: CRUD con laravel.

- Tema semana 10: Sistema de rutas y controladores.

- Tema semana 11: Validación de datos.

- Tema semana 12: Rest API con laravel.
